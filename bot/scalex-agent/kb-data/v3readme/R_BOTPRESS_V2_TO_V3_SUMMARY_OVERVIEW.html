Maxx Database v2 → v3

Summary for team · 24 Feb 2026 · Botpress package

What v3 is

v3 is the last structural change to the Maxx/Botpress tables. We have finalized the taxonomy content (per current best assumptions and task 13a IT updates) and locked the schema. The for siva v3 final package contains everything needed to implement against v3. Data population (AHT, rate validation, modifier quantification, etc.) is a separate workstream and does not change the schema.

v2 → v3 changes (what Botpress needs to implement)

What’s finalized (taxonomy)

  • TBL-001: 16 towers — no renames.
  • TBL-002: IT categories updated (task 13a): Application Development & Maintenance (ADM), End-User Services, IT Consulting & Advisory, Network & Telecommunications, Enterprise Software & Platforms (CAT-IT-009). Other towers unchanged.
  • TBL-004: IT: 8 new CAT-IT-009 service types (ERP Admin, CRM Platform Mgmt, SaaS License Mgmt, SAM, Software Compliance & Audit, Productivity Suite Admin, Enterprise Software Implementation, etc.). Service_Unit_Name / Service_Unit_Definition columns present; population deferred.
